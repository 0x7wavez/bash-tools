bash-tools

Bash scripts for Linux automation. Built from scratch, learning in public.
Scripts

    log-sysinfo.sh — Log system info to JSON
    config-reader.sh — Read config files + validate arguments
    log-analyzer.sh — Parse logs (grep, awk)
    file-watcher.sh — Monitor files + run actions
    port-scanner.sh — Scan ports (no external tools)

Why?

I'm building these as foundational skills for SecNode-X, a wireless security platform. Follow the journey: @0x7wavez
Structure

    /scripts/ — Executable scripts
    /lib/ — Shared functions
    /examples/ — Sample data + outputs
    /tests/ — Test scripts
    /docs/ — Detailed walkthroughs
